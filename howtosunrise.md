How to sunriseJS
=======


Getting startetd
---------

### Bind sunriseJS to the HTML

```
<div id="example" sunriseJS-app="my example app"></div>
```

and include sunrise JS at the end of the ```body``` tag

```
<body>
	<!-- other html stuff -->
	<script src="sunrise.js" type="text/javascript" async defer></script>
</body>
```
